Easy ... To Oracle - Requirements
Oracle
ETO uses the Oracle external table "pre-processor" feature. This feature is available from Oracle releases 10.2.0.5, 11.1.0.7 and 11.2.0.1.
Any edition of the database will do (Express, Standard, Enterprise). Since the "pre-processor" feature was developed for RAC it should also work in a RAC environment, but this has not been tested by me.

OS
ETO compiles on any 32- or 64 bit POSIX compliant operating system (LINUX, UNIX, Debian, Solaris, AIX, etcetera) and Windows.
It has been succesfully tested on Red-Hat 5 and 6, Ubuntu 10, 12 and 13 and Windows. See individual platform pages for required additional libraries.
