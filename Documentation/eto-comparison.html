Easy ... To Oracle - Comparison of data-integration methods
The are many ways to import data from a non-Oracle platform into an Oracle database. All with their advantages and disadvantages. Enough disadvantages to justify creating another piece of software to do the same job.

I can think of the following ways to import data into Oracle:
• Write to a flatfile, import flatfile
• Use an ODBC driver and Oracle Heterogenous Services
• Buy an Oracle Gateway module
• Use specialized ETL software (that may or may not use ODBC/JDBC)
• And now you can also use Easy To Oracle !

Note that I have not included stand-alone utilities for (one time) migration of data.
